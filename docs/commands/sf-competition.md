# sf competition

ワークショップ競技ツール（Day 5 ホバリングタイム競技）。

## 構文

```bash
sf competition <subcommand>
```

### サブコマンド

| サブコマンド | 説明 |
|-------------|------|
| `hover-time` | ホバリングタイム計測（競技タイマー） |
| `score` | 競技スコア・ランキング表示 |

## 使用例

```bash
sf competition hover-time
sf competition score
```
