# sf rc

リアルタイム RC（リモートコントロール）入力を StampFly に送信します。

## 構文

```bash
sf rc [a b c d]
```

### パラメータ（Tello SDK 互換）

| パラメータ | 範囲 | 説明 |
|-----------|------|------|
| `a` | -100 ~ 100 | 左右（roll） |
| `b` | -100 ~ 100 | 前後（pitch） |
| `c` | -100 ~ 100 | 上下（throttle） |
| `d` | -100 ~ 100 | ヨー（yaw） |

### モード

- **ワンショット:** `sf rc 0 50 0 0` - 単発送信
- **インタラクティブ:** `sf rc` - キーボード操作モード

## 使用例

```bash
sf rc 0 50 0 0
sf rc
```
