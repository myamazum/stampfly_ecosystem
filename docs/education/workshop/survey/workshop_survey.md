| L3 | LED 制御 |
| L4 | IMU センサ |
| L5 | レート P 制御 + 初フライト |
| L6 | システムモデリング |
| L7 | システム同定 |
| L8 | PID 制御 |
| L9 | 姿勢推定 |
| L10 | API リファレンス |
| L11 | 独自ファームウェア開発 |
| L12 | Python SDK プログラム飛行 |
| L13 | 精密着陸競技会 |

## Google Forms 作成手順

### 方法1: 手動作成
1. [Google Forms](https://forms.google.com) を開く
2. 「空白のフォーム」を作成
3. 上記の設問を順番に追加
4. セクション区切りを挿入（セクション1〜5）
5. 「設定」→「回答」→「メールアドレスを収集しない」を確認（匿名化）

### 方法2: Google Apps Script で自動作成
下記のスクリプトを Google Apps Script エディタに貼り付けて実行すると、
フォームが自動生成されます。

**手順:**
1. [Google Apps Script](https://script.google.com) を開く
2. 「新しいプロジェクト」を作成
3. 下記のコードを貼り付け
4. `createWorkshopSurvey()` を実行
5. 実行許可を承認
6. ログに出力されるフォームURLを開く
